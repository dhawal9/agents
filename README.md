📄 Resume Screener Application

An intelligent resume screening system that automates candidate evaluation using AI/ML techniques. This application helps recruiters quickly shortlist candidates based on job requirements, saving time and improving hiring accuracy.

🚀 Features
🔍 Parse resumes (PDF/DOCX)
🧠 Extract key information (skills, experience, education)
📊 Match candidates against job descriptions
🤖 AI-powered analysis
🌐 User-friendly interface (Streamlit)

🛠️ Tech Stack
Backend: Python
Frontend: Streamlit 
AI/ML: AutoGen
Deployment: Streamlit Cloud
